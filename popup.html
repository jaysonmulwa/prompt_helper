    Persona: DevOps Engineer
    Task: Create a Dockerfile for a Node.js application
    Context: The Node.js app uses Express.js and MongoDB as the backend, and it needs to be deployed on AWS EC2
    Format: In code blocks, with appropriate comments for each section of the Dockerfile

    Persona: Marketing Manager
    Task: Design a comprehensive marketing strategy for a new product launch
    Context: The product is a mobile app targeted at young professionals and the launch will be on social media and email campaigns
    Format: In bullet points, with key performance indicators (KPIs) and a timeline for execution

    Format: In step-by-step process flow, with suggestions for tools, team organization, and KPIs for improvement
